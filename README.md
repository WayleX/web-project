# web-project
#### This is gym app within the framework web course

## Up localy
#### Back-end
```
cd back-end/
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

* open http://localhost:8000/docs/
```
#### Front-end
```
cd front-end/
bun install
bun run dev

* open http://localhost:5173/
```
